# MBA Portfolio

A professionally designed dark-themed personal portfolio showcasing product management expertise, research-driven projects, and MBA-focused competencies. This portfolio reflects structured problem-solving, analytical thinking, and a strong foundation in product strategy and user-centric design.

> **Live Portfolio:** https://mba-portfolio-lime.vercel.app/
> **Local Preview:** Run `npm run dev` and open http://localhost:3000

---

## Tech Stack

| Layer      | Technology                      |
| ---------- | ------------------------------- |
| Framework  | Next.js (App Router, Turbopack) |
| Language   | TypeScript                      |
| Styling    | Tailwind CSS                    |
| Animations | Framer Motion                   |
| Icons      | Lucide React                    |
| Runtime    | React                           |

---

## Getting Started

### Prerequisites

* Node.js ≥ 18.x
* npm ≥ 9.x (or yarn / pnpm)

### Installation

```bash
# Clone the repository
git clone https://github.com/deepshikhaSantra/MBA-Portfolio.git

# Navigate into the project
cd MBA-Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at:

http://localhost:3000

---

## Production Build

```bash
npm run build
npm run start
```
